Features

  1. YOW Example and the software tools it describes are easy to learn and use. No server is required.
  2. The software tools work for one person or many people so an individual or group can create and maintain a web of static browser pages.
  3. YOW Example has a Wikipedia like presentation. Internal links are inline hyperlinks. External links are hyperlinks in footnotes.
  4. Like Wikipedia, YOW keeps a history of its changes so you can improve your own web as you change it.
  5. The software tools described are FOSS, Free Open Source Software.
  6. The described software tools are can be replaced one by one or in whole with other software. Replacement software can even be proprietary (non-FOSS) software.
