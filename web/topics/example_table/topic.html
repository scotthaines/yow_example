Example table

This table example uses an HTML table to hold the data and some classes in CSS to format the table.[1]

birds
# name date seen comments
1 Canada Goose 2014 08 30 This is a very common bird in parts of North America.
2 Emperor penguin This is a big bird and this is a long description. I put a lot of text here so it would wrap from one line to the next. The previous cell is left empty on purpose. Notice that the single lines of text in the other columns of this row are at the top of their cells. This is controlled by the td (table data) having a vertical-align attribute of top in the table_birds.css.
3 ThisIsARunOnWord 2010 01 02 As you narrow the browser display the run on word eventually displays with ellipsis.
4 Ostrich 2072 10 13 Striped rows help with reading ease. See tr:nth-child(even) in the table css.
