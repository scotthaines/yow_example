Requirements

Page creation and maintence:
Microsoft Windows operating systems are required because Tortoise Git only runs on them. Windows XP SP3 seems to be the oldest Microsoft Windows operating system supported by all of the software tools.
Browser compatability:
The goal is that pages display successfully on all main stream current or near browsers. Reasonably successfully here means everything is seen, all links work and format of the display is similar in all browsers. Microsoft Internet Explorer 9 works as well as Google Chrome Version 39. Results with Firefox and Safari are unknown.
Page display with various browser widths:
The full width of text displays so horizontal scrolling is not necessary to read. This is true of images as well for browser widths of 1024 pixels or greater. Images on narrower displays may be too wide depending on whether the image has been inserted in a fixed pixel dimensions rectangle or rectangle based on as a percent of the content area's width and the image's width to height ratio.

Browser widths of 1024 pixels or greater display fixed width centered pages. These pages completely fit a 1024 pixel wide browser with room to spare for a vertical scroll bar. This format limits the width of the single column of text to avoid the problem of losing track of the next line while reading very long lines on a wide display.

Browser widths less than 1024 pixels reduce the page width to fit the whole width in the view. This means text flows to fit the width.
