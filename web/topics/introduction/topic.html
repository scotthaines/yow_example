Introduction

YOW Example shows you how to write and maintain your own web of browser pages. If you do not want to start from scratch you can edit YOW Example's pages to turn them into your own pages.

YOW Example uses three programs to create and maintain browser pages. The first, Amaya, is a browser page editor. The second, Git, keeps a history of your changes so you can improve your pages over time. The third, Tortoise Git, puts the control of Git into right click popup menus in Microsoft Windows Explorer to make Git easy to use.
