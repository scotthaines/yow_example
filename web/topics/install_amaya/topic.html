Install Amaya

Amaya has only a few choices during install. Defaults choices during install work fine.

It is not necessary to make any file associations with Amaya. The easiest way to work with both Amaya and Git is by right clicking on files and folders in Windows Explorer.

Right click on a file in Windows Explorer to open with Amaya.
