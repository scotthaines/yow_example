YOW Example documents how to create Your Own Web of Wikipedia like static browser pages. The software tools described allow you to decide whether to make the browser pages you create public, private or proprietary. It's Your Own Web. You decide.

The software tools described are easy to learn and use. No server is required.

The software tools described work for one person or many people so an individual or group can create Your Own Web.

The software tools described are FOSS, Free Open Source Software. Also these software tools are used in a way that they can be replaced one by one or in whole with other software. The replacement software can even be proprietary (non-FOSS) software.
