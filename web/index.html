Your Own Web Example documents how to create and maintain Your Own Web (YOW) of static browser pages. The software tools described allow you make the browser pages public, private or proprietary. It's Your Own Web. You decide.

The software tools are easy to learn and use. No server is required.

The software tools work for one person or many people so an individual or group can create a web of static browser pages.

YOW Example has a Wikipedia like presentation. Internal links are inline hyperlinks. External links are hyperlinks in footnotes.

The software tools are FOSS, Free Open Source Software. Also these software tools are can be replaced one by one or in whole with other software. Replacement software can even be proprietary (non-FOSS) software.
